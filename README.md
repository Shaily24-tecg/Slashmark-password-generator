# Slashmark-password-generator

A simple command-line Password Generator application built using Python.

## Features
- Generate multiple random passwords  
- Include letters, numbers, and special characters  
- Password strength checker (Weak / Medium / Strong)  
- Save passwords automatically in a file  
- Stores output in a subfolder (`main/`)  
- Input validation for better user experience  

## How It Works
- User selects number of passwords  
- User defines length for each password (minimum 6)  
- Program generates secure random passwords  
- Each password is evaluated for strength  
- Output is displayed and saved to file  

## Requirements
- Python 3.x  

## Run the Project
```bash
python password-generator.py
